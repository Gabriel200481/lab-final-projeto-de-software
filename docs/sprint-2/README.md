# Sprint 2 (Lab03S02)

## Entregas

- Modelo ER: `uml/diagrama-er.puml`
- Estrategia de persistencia: Spring Data JPA + Hibernate
- CRUD inicial de aluno e empresa com front-end integrado ao back-end

## Evidencias

- Entidades JPA: `src/main/java/br/com/moedaestudantil/model`
- Repositorios: `src/main/java/br/com/moedaestudantil/repository`
- Endpoints CRUD:
  - `src/main/java/br/com/moedaestudantil/controller/AlunoController.java`
  - `src/main/java/br/com/moedaestudantil/controller/EmpresaController.java`
- Front-end estatico:
  - `src/main/resources/static/index.html`
  - `src/main/resources/static/app.js`

## Banco e inicializacao

- Configuracao local H2: `src/main/resources/application.yml`
- Dados seed: `src/main/resources/data.sql`
